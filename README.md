#流程指示

1. 查看天氣:開啟 App 後，首頁會自動顯示目前城市天氣。
   -啟動APP時會自動定位，若定位不在台灣內的縣市則會使用預設台北市的天氣。後續也可透過左上按鈕再次定位。
   -關注城市，即可通過關注頁面切換當前顯示天氣（右上角收藏）
   -查詢，也可自行輸入想查詢的城市
   -下拉會重新抓取即時天氣資訊（或透過右上角的重新載入）
   -根據城市的天氣切換背景（晴天、雨天、陰天）
3. 產生 AI 行程:切換到 AI 行程頁後，輸入你的出遊需求。
4. 確認行程:AI 行程生成後，畫面會出現行程預覽卡(於對話最上方)。
5. 產生穿搭建議:確認行程後，點擊"穿搭建議"。導至 AI Stylist 頁面。在此頁面點擊就會產生穿搭建議。(該功能因為圖片生成需額外花費故暫時無法正常運行)

P.S.:有部分檔案無法上傳可能導致功能缺失。
